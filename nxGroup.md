# NeXus 组使用说明

## nx 组（nxGroup）

nx 组（nxGroup）用于给粒子分配组 ID，并设置这个组的显示方式、颜色、速度、半径和质量等基础属性。

创建 nx 组（nxGroup）时，插件会在当前场景已有 nx 组中查找最大的组 ID，并给新组自动分配下一个编号。发射器（nxEmitter）可以在组（Groups）列表里引用 nx 组，让粒子进入指定组。

### 设置页（Section）

设置页（Section）切换 nx 组（nxGroup）当前显示的属性页。

当前 nx 组（nxGroup）只提供物体属性（Object Properties）页。组、映射和衰减页签只会出现在普通 NeXus 修改器上。

### 物体属性（Object Properties）

物体属性（Object Properties）显示 nx 组（nxGroup）的组 ID、显示模式、颜色模式、颜色，以及速度、半径和质量相关参数。

这些参数定义的是组本身提供给粒子的基础属性。是否使用这个组，取决于发射器或其他支持组引用的功能是否引用了它。

### 启用（Enabled）

启用（Enabled）控制 nx 组（nxGroup）是否参与当前 NeXus 管线。

关闭后，这个组对象不会作为有效组参与同步。引用它的发射器或其他功能需要它保持启用，才能正常使用组信息。

### 视口可见（Visible in Editor）

视口可见（Visible in Editor）控制 nx 组（nxGroup）在编辑器里的辅助显示是否可见。

这个开关影响编辑视图显示，不等同于关闭组本身。要停用组，应使用启用（Enabled）。

### 组 ID（Group ID）

组 ID（Group ID）是这个粒子组的数字编号。

创建新 nx 组（nxGroup）时，插件会自动给它分配当前场景中下一个可用编号。需要和其他设置或缓存数据保持对应时，可以手动确认这个编号。

### 显示模式（Display Mode）

显示模式（Display Mode）控制这个组的粒子在视口中如何显示。

可用显示方式包括点、方块、线、3D 盒、圆、金字塔、箭头、球体、轴向显示和隐藏。它影响显示外观，不改变粒子的物理计算。

### 颜色模式（Color Mode）

颜色模式（Color Mode）决定这个组的粒子颜色来源。

当前界面提供单色、渐变、着色器、物体颜色和噪波等模式。具体颜色能否按这些模式完整表现，还取决于后续显示和渲染流程。

### 颜色（Color）

颜色（Color）设置这个组的显示颜色。

在颜色模式使用单色或需要组颜色参与显示时，这个颜色会作为组的基础颜色。

### 速度（Speed）

速度（Speed）设置这个组的粒子基础速度。

它使用速度单位缩放同步到底层。数值越高，属于这个组的粒子初始速度越大。

### 速度变化（Speed Var）

速度变化（Speed Var）给速度（Speed）增加随机变化范围。

数值为 0 时，粒子使用稳定的基础速度。提高后，同组粒子的速度会产生差异。

### 半径（Radius）

半径（Radius）设置这个组的粒子物理半径。

它使用长度单位缩放同步到底层。半径会影响需要读取粒子大小的显示或模拟流程。

### 半径变化（Radius Var）

半径变化（Radius Var）给半径（Radius）增加随机变化范围。

数值为 0 时，同组粒子使用稳定半径。提高后，同组粒子的大小会产生差异。

### 质量（Mass）

质量（Mass）设置这个组的粒子质量。

质量会影响需要读取粒子质量的后续模拟或数据流程。

### 质量变化（Mass Var）

质量变化（Mass Var）给质量（Mass）增加随机变化范围。

数值为 0 时，同组粒子使用稳定质量。提高后，同组粒子的质量会产生差异。
