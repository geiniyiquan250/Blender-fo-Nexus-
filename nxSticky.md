# NeXus 粘性修改器使用说明

这份文档只说明粘性修改器（nxSticky）。它用于让粒子在接近目标表面时吸附上去，并按概率、距离、偏移、保持时间和衰减曲线控制粘附过程。

## 粘性修改器（nxSticky）

粘性修改器（nxSticky）不产生粒子。用户通常先创建能产生粒子的发射器（nxEmitter），再创建粘性修改器（nxSticky）。

这个修改器的核心输入是对象（Objects）列表。每个条目都引用一个网格（Mesh）、曲线（Curve）或曲线集合（Curves）对象，并让粒子尝试粘附到该对象表面或几何上。

每个对象条目都可以单独设置：

- 是否继承第一个条目的设置。
- 是否继承目标对象速度。
- 粘附概率、作用范围和容差。
- 是否只允许粘在外表面。
- 是否使用自定义偏移距离。
- 粘附保持时间和粘附衰减曲线。

### 启用（Enabled）

启用（Enabled）控制当前层级是否参与作用。

这个同名开关会出现在粘性修改器（nxSticky）本体和对象（Objects）列表项上。关闭某个对象条目后，粒子不会再尝试粘到这个对象上。

### 对象（Objects）

对象（Objects）是粘性修改器（nxSticky）的目标列表。

每个条目都指定一个可粘附的目标对象。当前允许的类型是网格（Mesh）、曲线（Curve）和曲线集合（Curves）。

### 活动对象索引（Active Object Index）

活动对象索引（Active Object Index）记录对象（Objects）列表当前选中的条目。

它决定下面显示和编辑的是哪一个条目的粘附参数。

### 添加对象（Add Object）

添加对象（Add Object）把一个可用对象加入对象（Objects）列表。

只有网格（Mesh）、曲线（Curve）和曲线集合（Curves）对象能作为有效粘附目标。

### 物体（Object）

物体（Object）指定当前条目引用的目标对象。

如果这里没有有效对象，这个条目不会参与粘附计算。

### 继承父级（Inherit Parent）

继承父级（Inherit Parent）让当前条目继承列表中第一个对象条目的设置。

开启后，当前条目自己的粘附参数区不会再单独编辑，而是直接跟随第一个条目的参数。它适合在多个对象需要完全相同粘附行为时减少重复设置。

### 继承速度（Inherit Speed）

继承速度（Inherit Speed）控制粒子粘附后是否继承目标对象的运动速度。

开启后，如果目标对象本身在移动，已粘附的粒子更容易跟着它一起动；关闭后，粒子更偏向只保持粘附，不额外继承对象速度变化。

### 概率（Probability）

概率（Probability）控制粒子在满足粘附条件时真正粘上的概率。

100% 表示只要命中条件就一定粘附；降低后，就会有一部分粒子即使进入范围也不会粘上。它适合做不均匀粘附、斑驳附着或随机残留效果。

### 范围（Range）

范围（Range）控制粒子距离目标表面多近时，允许进入粘附判定。

数值越大，可参与粘附的空间范围越宽；数值越小，粒子必须更贴近表面才可能粘上。

### 容差（Tolerance）

容差（Tolerance）控制粘附判定时允许的距离容差。

它和范围（Range）一起决定“多近才算接触到表面”。如果粒子偶尔明明贴近了却没有粘上，可以适当提高容差（Tolerance）。

### 随对象旋转（Rotate With Object）

随对象旋转（Rotate With Object）控制已粘附粒子是否跟随目标对象一起旋转。

开启后，目标对象转动时，粒子会更明显地一起转；关闭后，粒子主要保持位置粘附，不额外跟着对象姿态转动。

### 仅外部（Outside Only）

仅外部（Outside Only）控制粒子是否只允许粘在目标表面的外侧。

开启后，更适合做“粘在外壳表面”的效果；关闭后，粒子可能在更宽松的几何条件下粘附。

### 自定义偏移（Custom Offset）

自定义偏移（Custom Offset）控制是否启用额外的表面偏移距离。

开启后，下面的距离（Distance）才会生效，用来让粒子和表面之间保留一定间隔。

### 距离（Distance）

距离（Distance）只在自定义偏移（Custom Offset）开启时生效。

它控制粒子粘附后距离目标表面的偏移量。数值越大，粒子离表面越远；为 0 时表示贴在表面上。

### 粘附时间（Sticky Time）

粘附时间（Sticky Time）控制粒子保持粘附状态多久。

数值为 0 时表示无限期保持粘附；大于 0 时，粒子会在这段时间结束后脱离。它适合做“先粘住再掉落”或“停留一段时间再释放”的效果。

### 粘附衰减（Sticky Falloff）

粘附衰减（Sticky Falloff）是配合粘附时间（Sticky Time）使用的曲线。

它控制粒子在粘附期间，粘附强度如何随时间变化。默认曲线前段保持较强，后段快速下降，所以很适合“先稳稳贴住，再逐渐失去附着力”的效果。

---

## 使用建议

### 多对象共用同一套粘附规则

如果多个表面都要用同样的粘附行为，可以把第一个对象条目调好，再给后面的条目开启继承父级（Inherit Parent）。

### 做随机残留效果

先降低概率（Probability），再配合适中的范围（Range）和容差（Tolerance），可以得到不是所有粒子都会粘上的自然残留效果。

### 做“沾上后再掉落”

把粘附时间（Sticky Time）设成有限值，并调整粘附衰减（Sticky Falloff）曲线，让粒子在后段逐渐失去附着力。
